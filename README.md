## 📌 О проекте

**Telegram Schedule Bot** - это Telegram‑бот для удобного просмотра
расписания занятий.\
Проект разработан в рамках **индивидуального проекта** и предназначен
для автоматизации доступа к расписанию через Telegram.

Бот позволяет пользователям быстро получать актуальную информацию о
занятиях, а администраторам - управлять системой через удобный
интерфейс.

------------------------------------------------------------------------

# 🚀 Возможности

## 👤 Пользователь

-   регистрация в системе
-   выбор класса / группы
-   просмотр расписания
-   быстрые кнопки для навигации
-   удобное взаимодействие через Telegram

## 🛠 Администратор

-   управление пользователями
-   управление группами
-   редактирование расписания
-   просмотр базы данных пользователей
-   назначение администраторов

------------------------------------------------------------------------

# 🧠 Архитектура проекта

Проект построен по **модульной архитектуре**, что упрощает поддержку и
масштабирование.

    src/
    │
    ├── configs/          # конфигурация проекта
    │   ├── database.py
    │   └── loader.py
    │
    ├── handlers/         # обработчики команд
    │   ├── admin/
    │   ├── user/
    │   └── registration.py
    │
    ├── keyboards/        # клавиатуры Telegram
    │   ├── inline.py
    │   └── reply.py
    │
    ├── middlewares/      # middleware
    │   └── logger.py
    │
    ├── states/           # FSM состояния
    │
    └── main.py           # запуск бота

# 📦 Установка

## 1️⃣ Клонирование репозитория

``` bash
git clone https://github.com/Sad0wW/telegram-bot-schedule
cd telegram-bot-schedule
```

## 2️⃣ Установка зависимостей

``` bash
pip install -r requirements.txt
```

## 3️⃣ Настройка конфигурации

Создайте файл `config.json`:

``` json
{
    "telegram": {
        "token": "YOUR_BOT_TOKEN",
        "admin_id": 123456789
    }
}
```

------------------------------------------------------------------------

# ▶️ Запуск

    python src/main.py

После запуска бот начнет работать через **Telegram Bot API (long
polling)**.

------------------------------------------------------------------------

# 📊 Пример работы
